# Calculator

A simple web-based calculator that performs basic arithmetic operations like addition, subtraction, multiplication, and division.

## Features

* Perform basic calculations
* Clean and simple user interface
* Responsive design
* Works directly in the browser

## Technologies Used

* HTML
* CSS
* JavaScript

## Installation

1. Clone the repository

git clone [git@github.com](mailto:git@github.com):prince32456/calculator.git

2. Navigate to the project folder

cd calculator

3. Open the project

Open `index.html` in your browser.

## Usage

* Click the number buttons to enter values
* Select an operator (+, −, ×, ÷)
* Press `=` to see the result
* Use `C` to clear the screen
